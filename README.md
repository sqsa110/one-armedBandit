# 老虎机抽奖
## 初始化参数
**ele**  //外层元素,必要参数

**list** //中奖列表  {src,type}  src 图片链接  type 奖品标记 必要参数

**type** //-1为不中奖  中奖设置type值  默认不中奖

**allStop**   //同步停止，异步停止 默认异步

**stopTime**  //停止时间  默认3000毫秒

## 内置方法

**init**  //开始程序，并默认stopTime 时间结束，可以传入一个回调函数，并接收type

**star**  //开始运转,并不会关闭

**stop**  //结束抽奖，可以传入一个回调函数，并接收type

**setType** //设置中奖值

**getType** //读取中奖值
